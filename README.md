# aegistalk-core
AegisTalk — защищенный E2EE мессенджер на Go и Flutter. Реализация Double Ratchet протокола для PFS. Собственный транспорт (TLS/WebSockets) в обход почтовых спам-фильтров. Высокая производительность, сериализация через Protobuf и надежное хранение сессий в PostgreSQL. Техническое решение для стабильного и безопасного обмена данными.
